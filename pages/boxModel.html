Teoria de cajas (Box model)

There are two types of box model: block and inline-block. - block takes the full width of the parent element. - inline-block takes the width of the content. Some elements have by default block, (h1, h2, etc. p ...) And some others have inline-block, (b, span, etc.)

Segundo H2 puesto como inline block

Restricciones

A los elementos en linea:
  • No les puedo ajustar la altura, ni otras propiedades
    height: 400px;
    width: 400px;
    NO funcionará
A los elementos en bloque
  • Sí les puedo ajustar la altura, y otras propiedades
    height: 400px;
    width: 400px;
    SÍ funcionará
